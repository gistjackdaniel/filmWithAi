# SceneForge UI 와이어프레임

## 개요
이 문서는 SceneForge 애플리케이션의 주요 화면들의 와이어프레임을 정의합니다.

## 1. 로그인 화면

### 1.1 전체 레이아웃
```
┌─────────────────────────────────────────────────────────┐
│                    SceneForge                          │
│                                                       │
│                                                       │
│              🎬 AI 영화 제작 툴                        │
│                                                       │
│                                                       │
│              ┌─────────────────┐                      │
│              │  Google 로그인   │                      │
│              └─────────────────┘                      │
│                                                       │
│                                                       │
│              시놉시스만 입력하면                        │
│              AI가 자동으로 스토리와 콘티를 생성합니다    │
│                                                       │
│                                                       │
└─────────────────────────────────────────────────────────┘
```

### 1.2 구성 요소
- **헤더**: SceneForge 로고 및 제목
- **메인 콘텐츠**: 
  - 앱 소개 문구
  - Google 로그인 버튼
  - 기능 설명 텍스트
- **푸터**: 버전 정보

### 1.3 반응형 디자인
- **데스크톱**: 중앙 정렬, 큰 버튼
- **태블릿**: 적절한 여백 조정
- **모바일**: 전체 화면 활용

## 2. 대시보드 (메인 화면)

### 2.1 전체 레이아웃
```
┌─────────────────────────────────────────────────────────┐
│ [SceneForge] [사용자명 ▼] [로그아웃]                   │
├─────────────────────────────────────────────────────────┤
│                                                       │
│  프로젝트 목록                    [+ 새 프로젝트]      │
│                                                       │
│  ┌─────────────┐ ┌─────────────┐ ┌─────────────┐     │
│  │ 프로젝트 1   │ │ 프로젝트 2   │ │ 프로젝트 3   │     │
│  │ 제목: 로맨스 │ │ 제목: 액션   │ │ 제목: 스릴러 │     │
│  │ 생성일: ...  │ │ 생성일: ...  │ │ 생성일: ...  │     │
│  └─────────────┘ └─────────────┘ └─────────────┘     │
│                                                       │
│  최근 활동                                        │
│  • 프로젝트 1 콘티 생성 완료                          │
│  • 프로젝트 2 스토리 생성 완료                        │
│                                                       │
└─────────────────────────────────────────────────────────┘
```

### 2.2 구성 요소
- **헤더**: 
  - 로고 및 네비게이션
  - 사용자 메뉴 (프로필, 로그아웃)
- **메인 콘텐츠**:
  - 프로젝트 목록 (카드 형태)
  - 새 프로젝트 생성 버튼
  - 최근 활동 목록
- **사이드바** (선택적):
  - 빠른 액션 메뉴
  - 필터 옵션

## 6. 반응형 디자인 가이드

### 6.1 브레이크포인트
- **모바일**: 320px - 768px
- **태블릿**: 768px - 1024px
- **데스크톱**: 1024px 이상

### 6.2 모바일 최적화
- 타임라인: 세로 스크롤로 변경
- 카드: 전체 너비 활용
- 버튼: 터치 친화적 크기 (44px 이상)

### 6.3 태블릿 최적화
- 사이드바: 접을 수 있는 형태
- 타임라인: 중간 크기 카드
- 모달: 적절한 여백 유지
