# 🎬 시간 기반 타임라인 UI 구현 태스크 목록

## 📋 개요
Adobe Premiere Pro와 같은 시간 기반 타임라인 UI 구현
- 타임라인은 전체 동영상의 길이를 나타냄
- 각 씬이 차지할 시간만큼 수평 스크롤의 공간을 차지
- 시간 스케일과 줌 기능 제공
