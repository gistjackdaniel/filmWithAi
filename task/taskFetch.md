## 📝 **체크리스트 사용법**

- [ ] 완료된 태스크는 체크박스를 체크
- [ ] 진행 중인 태스크는 `[진행중]` 표시
- [ ] 블로커가 있는 태스크는 `[블로커]` 표시
- [ ] 완료 시 날짜와 담당자 기록

## 🎯 **성공 기준**

- [ ] 프로젝트 생성 → 콘티 생성 플로우가 단일 경로로 통일
- [ ] 시놉시스 입력이 선택적이 되면서도 프로젝트 관리 용이성 확보
- [ ] 기존 기능들이 새로운 플로우에서 정상 작동
- [ ] 사용자 경험이 개선되고 혼란 요소 제거
